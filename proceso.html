Proceso



A continuación, se describirán los pasos a seguir que hubo para llegar a una imagen,
no se subirán las encuestas realizadas respetando la privacidad e información personal
de cada uno de los encuestados.

1. Encuesta realizada a un grupo de personas, se busca tener distintas relaciones y
descripciones de las emociones desde objetos, formas, colores y demás posibles imágenes
que existen en nuestra realidad.

2. Descripciones relacionadas a tres emociones específicas, que son las que se trabajaron en
este proyecto, la tristeza y el Miedo.

3. Se empiezan a hacer relaciones entre las descripciones para encontrar similitudes.

4. Se construyen imágenes de acuerdo con las descripciones.

















Ejemplo



• Tristeza: Nada, lúgubre, pesimismo, lento, vacío. Negativo. Lágrimas, dolor, frio.
Larga. Azul.

• Miedo: Sudoroso, movimiento, apresurado, lento, congelado. Desconocido, conocido.
Preocupación, escalofrió. Siempre. Vacío. Profundo, pequeño, gigante. Negro.
